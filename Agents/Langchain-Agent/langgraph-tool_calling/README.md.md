# LangGraph-Agent  Internet Search

## Install libraries:
* langchain 
* langgraph 
* langchain-groq 
* duckduckgo-search 
* beautifulsoup4 
* gradio
* python-dotenv

## Set `.env`
```code
GROQ_API_KEY = Your_secret_key
SERPER_API_KEY = Your_secret_key
```
